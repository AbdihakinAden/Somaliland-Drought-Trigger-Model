# Somaliland Drought Trigger Model - Model Development Repository

## Repository Scope
**⚠️ This repository contains ONLY the model development component of the thesis:**

> "Building the Foundation: A Trigger Model for Anticipatory Drought Action in Somaliland"

**What's included:**
- Drought trigger model development and validation
- SPI-3 data processing and standardization
- Dry percentage extraction
- Model performance evaluation
- **Sample data**: MAM and OND seasons (2020-2023)

**What's NOT included:**
- Full impact database (85 records from 2000-2023)
- Stakeholder co-design process
- Complete operational framework implementation
- Policy recommendations

## Sample Data Coverage
- **Seasons**: MAM (March-April-May) and OND (October-November-December)
- **Period**: 2020-2023
- **Regions**: All 5 Somaliland administrative regions

## Research Context
This work addresses the "science-action gap" in drought early warning by developing and validating three progressive drought detection models for Somaliland's pastoral systems.
